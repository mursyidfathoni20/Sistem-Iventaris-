# Sistem-Iventaris-
Sistem Inventaris Barang berbasis web digunakan untuk mengelola data inventaris secara mudah. Sistem memiliki fitur login, CRUD data barang, serta transaksi barang masuk dan barang keluar yang memperbarui stok secara otomatis sehingga pengelolaan inventaris menjadi lebih cepat, rapi, dan akurat.
